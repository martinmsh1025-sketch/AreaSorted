# Google Search Console Checklist

## 1. Add Property

- Go to Google Search Console
- Add your production domain
- Prefer `Domain property` if possible

## 2. Verify Ownership

Recommended options:

- DNS verification at your domain provider
- Or HTML tag verification if needed

## 3. Submit Sitemap

Submit:

- `https://your-domain.com/sitemap.xml`

## 4. Check Indexing

After launch, review:

- Pages indexed
- Pages excluded
- Any crawl errors
- Any canonical issues

## 5. Monitor Queries

In Performance, check:

- impressions
- clicks
- CTR
- average position
- top queries
- top pages

## 6. Early Priority Pages

Watch these first:

- `/`
- `/services`
- `/pricing`
- `/faq`
- `/contact`
- `/london/camden`
- `/london/islington`
- `/london/westminster`
- `/london/hackney`
- `/london/lambeth`

## 7. Use URL Inspection

Use URL Inspection to request indexing for:

- homepage
- services page
- pricing page
- FAQ page
- each borough landing page

## 8. What Success Looks Like Early On

In the first few weeks, focus on:

- pages getting indexed cleanly
- brand search appearing correctly
- long-tail London service queries starting to get impressions
- service and borough pages collecting clicks
