# SEO Roadmap

## Current Position

AreaSorted already has a solid technical base for SEO:

- Next.js metadata on key marketing pages
- `sitemap.xml` and `robots.txt`
- FAQ structured data
- Organization and local business structured data
- Service, pricing, FAQ, legal, and contact content already indexable

This gives the project a realistic chance to rank for long-tail and local-intent queries, especially across London service combinations.

## What To Target First

Do not start by chasing ultra-competitive broad terms like `cleaners london`.

Start with:

- service + borough combinations
- service + need/state combinations
- comparison / cost / how-it-works searches

Examples:

- end of tenancy cleaning camden
- handyman islington
- pest control westminster
- furniture assembly hackney
- waste removal lambeth
- garden maintenance southwark
- local cleaning service pricing london
- how does areasorted booking work

## Service SEO Structure

For each major category, create or deepen one indexable landing page with:

1. What the service covers
2. Common jobs customers book
3. What changes the price
4. What happens after booking
5. London coverage wording
6. Internal links to pricing, FAQ, contact, and quote

Priority order:

1. Cleaning
2. Handyman
3. Pest control
4. Waste removal
5. Furniture assembly
6. Garden maintenance

## Borough Landing Plan

Start with 5 borough pages only.

Recommended first batch:

1. Camden
2. Islington
3. Westminster
4. Hackney
5. Lambeth

Each borough page should include:

- borough-specific intro
- covered service categories
- common property / job types in that borough
- pricing expectations
- booking process summary
- FAQ snippets for that borough
- CTA to continue booking

Suggested URL pattern:

- `/london/camden`
- `/london/islington`
- `/london/westminster`
- `/london/hackney`
- `/london/lambeth`

## Before Public Launch

Finish these first:

- replace all legal/content placeholders
- finalise privacy and cookie implementation wording
- keep payment hold / capture wording consistent everywhere
- add Google Search Console
- add GA4

## Traffic Measurement Setup

Use both tools:

- `Google Analytics 4`
  - sessions
  - engaged sessions
  - conversions
  - traffic channels
- `Google Search Console`
  - impressions
  - clicks
  - queries
  - indexed pages
  - search ranking by page/query

Search Console is the more important SEO tool. GA4 tells you what users do after they arrive.

## Success Benchmarks For Early SEO

In the first phase, success is:

- pages are indexed cleanly
- brand searches show the site correctly
- long-tail London queries start getting impressions
- service/pricing/FAQ pages begin collecting clicks

Do not judge SEO only by raw traffic in the first few weeks.
