# London Cleaning Platform - Brand Identity v1

## Working Brand

- Primary working name: `Alder London`
- Service descriptor: `Alder London Cleaning`
- Recruitment descriptor: `Work with Alder London`

## Brand Position

`Alder London` should feel like a modern London service company with real operational standards.

It should communicate:
- trust
- order
- discretion
- local professionalism
- warm reliability

## Visual Direction

The visual language should sit between:
- a premium home services brand
- an editorial lifestyle brand
- a practical operations-led cleaning company

It should not feel like:
- a generic SaaS startup
- a discount cleaner leaflet
- a fake luxury concierge company

## Color Palette

### Core Palette

- `Alder Ink` - `#20312C`
  - primary dark color
  - use for headings, strong buttons, anchors

- `Stone Wash` - `#F4F0E8`
  - primary background
  - use for page background and softer sections

- `Linen` - `#E7DED0`
  - secondary background / card tint
  - use for panels and subtle dividers

- `Moss Grey` - `#6D7A72`
  - supporting muted text / UI accents
  - use for secondary copy and meta details

- `Brass Clay` - `#A8744F`
  - accent color
  - use for highlights, hover accents, small emphasis only

### Utility Palette

- `Charcoal` - `#1A1A1A`
- `White` - `#FFFFFF`
- `Soft Line` - `#D8D0C4`
- `Success` - `#2F6B4F`
- `Warning` - `#A06A2C`
- `Error` - `#9A3E34`

## Color Usage Rules

- default page background should be `Stone Wash`, not pure white
- use `Alder Ink` as the brand anchor, not bright navy
- use `Brass Clay` sparingly; it is an accent, not a main color
- key CTAs should mostly be dark solid buttons with light text
- avoid too many colored cards on one page

## Typography

### Recommended Pairing

- Headline font: `Fraunces`
- Body font: `Manrope`

### Why

- `Fraunces` gives the brand an editorial, premium, non-generic voice
- `Manrope` keeps forms, UI, and body text modern and clear

### Typography Roles

- `Display / H1`: Fraunces, large, slightly tight line-height
- `H2 / H3`: Fraunces or bold Manrope depending on section density
- `Body`: Manrope regular / medium
- `Buttons / labels / metadata`: Manrope medium

### Typography Rules

- use sentence case, not all caps for major headings
- keep heading lines short and confident
- body copy should be clean and readable, not decorative
- avoid tiny light-grey text

## Logo Direction

### Phase 1 Logo

Start with a wordmark.

Primary lockup:
- `Alder London`

Secondary lockup:
- `Alder London Cleaning`

### Logo Style

- elegant serif or serif-sans combination
- balanced spacing
- understated, calm, premium

### Avoid

- broom icon
- cartoon sparkle icon
- roofline / house clichĂ© unless extremely subtle
- fake luxury crest

### Optional Monogram

- `AL`
- use later for favicon or social avatar

## Photography Direction

Use imagery that feels:
- real
- bright but soft
- tidy and modern
- domestic and London-relevant

Preferred image themes:
- calm, well-kept interiors
- natural daylight
- subtle lifestyle details
- professional service moments without staged smiles

Avoid:
- cheap stock images of spray bottle close-ups
- exaggerated before/after shots
- photos that look like agency call-center ads

## UI Style Rules

### Layout

- generous spacing
- clear section rhythm
- avoid crowded blocks
- use mixed-width sections, not all same-width card rows

### Surfaces

- slightly tinted cards and panels
- soft borders instead of heavy shadows
- occasional textured or tonal background sections

### Buttons

Primary button:
- dark `Alder Ink` background
- light text
- slightly rounded corners

Secondary button:
- transparent or linen background
- dark border and text

### Inputs

- calm, clean form styling
- strong label visibility
- subtle border, warm background, good spacing
- mobile-friendly field height

### Icons

- use simple line icons only
- no emoji-like icons

## Homepage Visual Language

### Hero Direction

Hero should feel like:
- a polished London home service brand
- clear and conversion-focused
- built for both customers and job applicants

Suggested hero structure:
- left: headline, supporting text, trust bullets, CTA buttons
- right: quote panel or layered brand panel

Suggested headline style:
- calm and assured, not salesy

Example direction:
- Trusted cleaning across London, with verified professionals

### Section Rhythm

Recommended homepage order:
- hero
- trust strip
- how it works
- service cards
- why choose us
- cleaner recruitment section
- area coverage
- testimonials
- FAQ
- final CTA

### Background Strategy

- alternate between `Stone Wash`, `White`, and subtle tinted sections
- use quiet contrast, not hard dark/light blocks everywhere

## Brand Copy Style

Customer-facing copy should sound:
- direct
- reassuring
- competent

Cleaner-facing copy should sound:
- respectful
- legitimate
- structured

Avoid copy that sounds:
- too cheap
- too pushy
- too startup-jargony

- brand: `Alder London`
- visual tone: premium local London service brand
- palette: stone, deep green, muted brass
- typography: `Fraunces` + `Manrope`
